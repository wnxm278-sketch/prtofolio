# 📌 Project Overview

This project demonstrates a complete Data Warehouse (DWH) solution for Airbnb listings across European cities.

The project follows the Medallion Architecture:

- Bronze Layer
- Silver Layer
- Gold Layer

After building the Data Warehouse, an interactive Power BI dashboard was created to analyze Airbnb listings and generate business insights.

# 📊 Dashboard Pages

This page provides a general overview of the Airbnb dataset using KPIs and visualizations.

### KPIs

- Total Listings
- Average Price
- Average Bedrooms
- Average Guest Satisfaction

### Charts

- Average Price by City
- Listings by Room Type
- Cleanliness Rating by City
- Airbnb Locations Map
